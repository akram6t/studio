# **App Name**: ExamPrep

## Core Features:

- Exams Page: Display trending exams and category-wise exams with a visually appealing, responsive design.
- Exam Detail Page: Display basic exam information with tabs for Overview, Mock Tests, Tests, Previous Papers, Content, and Quizzes.
- Overview Tab: Show exam overview content from MDX files.
- Mock Tests, Tests, Prev. Papers Tabs: Display mock tests, tests and previous papers in a list view with details like title, duration, marks, and number of questions using data from JSON.
- Content Tab: Show PDF, PPT, and blog cards in a responsive grid view.
- Quizzes Tab: Display MCQ-based quizzes in a responsive grid view.
- Login/Signup Pages: Implement basic login and signup pages with fake authentication.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and knowledge. Inspired by the seriousness of exam preparation, avoiding cliches like bright or obvious colors.
- Background color: Light gray (#F5F5F5) for the light theme and Dark gray (#212121) for the dark theme.
- Accent color: A teal-leaning light blue (#26A69A) provides a calm, trustworthy pop.
- Body font: 'Inter', a sans-serif font, to ensure readability and a clean modern look.
- Headline font: 'Space Grotesk', a sans-serif font, for titles to convey modernity.
- Use simple, clear icons from a library like FontAwesome or Remix Icon to represent exam categories and features.
- Implement a responsive, mobile-first layout using Tailwind CSS grid and flexbox. Ensure consistent spacing and alignment across all devices.
- Use subtle transitions and animations for loading states and UI interactions to enhance the user experience without being distracting.
